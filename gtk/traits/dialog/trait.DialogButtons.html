Pseudo-variadic array of buttons

It's implemented for fixed-sized arrays [(&str, i32); N] and [(&str, ::ResponseType); N] (N <= 16) to allow passing variable numbers of buttons to some dialog methods.

Dialog::with_buttons(title, parent, flags,
                     [("Ok", ResponseType::Accept), ("Cancel", ResponseType::Cancel)]);
